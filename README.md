# DFe
DLL desenvolvida com o intuito de facilitar a integração dos Softwares ERP com o sistema de documentos fiscais eletrônicos da SEFAZ (NFe, NFCe, MDFe, CTe, NFSe, GNRE, EFDReinf, etc.).
